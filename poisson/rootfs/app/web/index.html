Browser Extension Setup

The Poisson browser extension generates noise traffic from your real browser, using your actual cookies and fingerprint. This makes fake traffic indistinguishable from your real browsing.

  1. Download Extension (.zip)
  2. Extract the zip file to a folder on your computer
  3. Open Chrome and go to chrome://extensions
  4. Enable Developer mode (toggle in top-right corner)
  5. Click Load unpacked and select the extracted folder
  6. Click the Poisson fish icon in your browser toolbar
  7. Enter your Home Assistant URL and a Long-Lived Access Token Create a token in HA → Profile → Long-Lived Access Tokens
  8. Click Connect — you're done!
